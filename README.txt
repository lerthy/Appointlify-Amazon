# Appointly

Appointly is a web-based appointment scheduling platform designed to help businesses manage bookings, staff availability, and customer interactions with ease. It allows customers to book services online while giving businesses full control over employees, services, and schedules.

## Features

- ✅ Customer self-service booking
- ✅ Business profile management
- ✅ Service creation with duration and pricing
- ✅ Real-time availability & calendar integration
- ✅ Appointment reminders
- ✅ Admin dashboard for managing appointments and team
- ✅ Mobile and desktop responsive

## Tech Stack

- **Frontend**: React.js 
- **Backend**: Node.js
- **Database**: PostgreSQL Supabase
- **Deployment**: Netlify

## Getting Started

### Prerequisites

- Node.js >= 18.x
- PostgreSQL >= 13
- Git
